# Podcast Converter Skill

一个为 AI Agent 设计的自动化播客下载与转换工具。

## ✨ 特性
- **跨平台支持**：支持 Windows、macOS 和 Linux。
- **一键下载**：自动处理小宇宙等平台的播客链接。
- **自动转码**：自动使用 ffmpeg 转换为 192kbps MP3。
- **智能元数据**：自动识别并提取播客节目名称。

## 📦 安装

```bash
gemini skills install https://github.com/charliebu/podcast-converter-skill.git --scope user
```

## ⚙️ 依赖
请确保您的系统已安装：
- [yt-dlp](https://github.com/yt-dlp/yt-dlp)
- [ffmpeg](https://ffmpeg.org/)
- Python 3.10+
